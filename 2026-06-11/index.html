Vectors in \(\mathbb{R}^n\). A vector is an ordered \(n\)-tuple \(\mathbf{v} = (v_1, v_2, \dots, v_n),\; v_i\in\mathbb{R}\). The set \(\mathbb{R}^n\) is a vector space over \(\mathbb{R}\): closed under componentwise addition and scalar multiplication.

The Dot Product. The canonical inner product on \(\mathbb{R}^n\) is the bilinear, symmetric, positive-definite map

\[ \mathbf{u}\cdot\mathbf{v} = \sum_{i=1}^{n} u_i v_i. \]

Derived from the law of cosines in \(\mathbb{R}^2\): \(\|\mathbf{u}-\mathbf{v}\|^2 = \|\mathbf{u}\|^2 + \|\mathbf{v}\|^2 - 2\|\mathbf{u}\|\|\mathbf{v}\|\cos\theta\). Expanding the left sum and canceling \(\sum u_i^2 + \sum v_i^2\) yields \(\mathbf{u}\cdot\mathbf{v} = \|\mathbf{u}\|\|\mathbf{v}\|\cos\theta\).

Cauchy-Schwarz Inequality. For all \(\mathbf{u},\mathbf{v}\in\mathbb{R}^n\),

\[ |\mathbf{u}\cdot\mathbf{v}| \le \|\mathbf{u}\|\;\|\mathbf{v}\|, \]

with equality iff \(\mathbf{u}\) and \(\mathbf{v}\) are linearly dependent. Proof: Consider \(f(t) = \|\mathbf{u} - t\mathbf{v}\|^2 = \|\mathbf{u}\|^2 - 2t(\mathbf{u}\cdot\mathbf{v}) + t^2\|\mathbf{v}\|^2 \ge 0\). The discriminant \(\Delta = 4(\mathbf{u}\cdot\mathbf{v})^2 - 4\|\mathbf{u}\|^2\|\mathbf{v}\|^2\) must be \(\le 0\), giving the inequality. Equality occurs when \(\Delta = 0\), implying a double root \(t_0\) such that \(\mathbf{u} = t_0\mathbf{v}\). ∎

Worked Example 1. \(\mathbf{u} = (1, -2, 3),\; \mathbf{v} = (4, 0, -1)\): \(\mathbf{u}\cdot\mathbf{v} = 1\), \(\|\mathbf{u}\| = \sqrt{14}\), \(\|\mathbf{v}\| = \sqrt{17}\). Cauchy-Schwarz holds: \(1 \le \sqrt{14\cdot17}\). \(\cos\theta = 1/\sqrt{238} \approx 0.0648\).

Worked Example 2 (Sharpness). \(\mathbf{u} = (2, -1),\; \mathbf{v} = (-4, 2) = -2\mathbf{u}\). Dot product: \(-10\); norms: \(\sqrt{5},\;2\sqrt{5}\). Bound saturates: \(10 = (\sqrt{5})(2\sqrt{5})\).

The Perceptron. Biologically motivated by the neuron (dendrites → soma → axon), the McCulloch-Pitts neuron (1943) formalized a threshold logic unit \(y = \mathbb{1}(\sum w_i x_i \ge \theta)\) — but with no learning rule. Rosenblatt's perceptron (1958) introduced the critical innovation: weights adapt from data.

Learning Rule. The perceptron computes \(\hat{y} = \operatorname{sgn}(w^\top x)\) with \(y^{(i)} \in \{-1,+1\}\). The perceptron criterion loss \(\mathcal{L}(w) = \max(0, -y w^\top x)\) gives gradient

\[ \nabla_w \mathcal{L} = \begin{cases} 0 & \text{if } y w^\top x \ge 0 \\ -y x & \text{otherwise} \end{cases} \]

yielding the update \(w_{t+1} = w_t + \eta\, y^{(i)} x^{(i)}\) on each misclassified example.

Perceptron Convergence Theorem (Novikoff, 1962). If data is linearly separable with margin \(\gamma > 0\) such that \(y^{(i)}(w^{*\top}x^{(i)}) \ge \gamma\) and \(R = \max_i\|x^{(i)}\|\), the perceptron converges in at most \(R^2/\gamma^2\) updates. Proof sketch: Two inductive bounds — (1) \(w_t^\top w^* \ge t\eta\gamma\) from the margin condition, and (2) \(\|w_t\|^2 \le t\eta^2 R^2\) from the update norm. Combining via Cauchy-Schwarz: \(t\eta\gamma \le \sqrt{t}\,\eta R \implies t \le R^2/\gamma^2\). ∎

Failure on XOR. Minsky and Papert (1969) proved the perceptron cannot represent the XOR function \(f(x_1, x_2) = x_1 \oplus x_2\) because no single linear surface separates the four points. This limitation was widely (and controversially) interpreted as demonstrating fundamental neural network inadequacy, contributing to the first AI Winter.

Legacy. The XOR problem is solved by multi-layer perceptrons with hidden layers composing nonlinearities — each hidden neuron learns a separate decision boundary, the output layer combines them into a convex polytope. This insight underlies backpropagation, deep learning, and every modern architecture from ResNets to transformers.

The field is converging toward an "MLOps 2.0" that treats LLMs as stateful distributed systems rather than stateless prediction APIs.
